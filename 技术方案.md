# 技术方案

## 一、 开发环境与基础构建

| 工具/库 | 版本建议 | 用途与说明 |
| :--- | :--- | :--- |
| **JDK** | **21 (LTS)** | 无 |
| **Maven** | **3.9.x** | 项目构建和依赖管理。 |
| **IntelliJ IDEA** | **Ultimate** | **必选**。 |
| **Lombok** | **最新版** | **推荐**。这是一个 Maven 依赖。它可以让你通过注解（`@Data`）自动生成 Getters/Setters/Constructor。 |

## 二、 后端核心

不使用 Spring，我们需要手动组装轻量级组件。

| 工具/库 | 依赖名称 (Maven) | 用途与说明 |
| :--- | :--- | :--- |
| **Web 服务器** | **Apache Tomcat 11** | 注意：需在 `pom.xml` 中引入 `jakarta.servlet-api`。 |
| **数据库** | **MariaDB** | 完全兼容 MySQL，轻量稳定。 |
| **数据库驱动** | **mariadb-java-client** | **必须补充**。这是 Java 连接 MariaDB 的桥梁（JDBC Driver）。 |
| **ORM 工具** | **Apache Commons DbUtils** | 它比原生的 JDBC 简洁，又没有 MyBatis/Hibernate 那么繁琐的配置。主要用 `QueryRunner` 类。 |
| **连接池** | **HikariCP** | **建议补充**。虽然 DbUtils 可以直连，但为了稳定性（防止数据库断开），建议加一个连接池，HikariCP 是目前最快最简单的。 |
| **工具包** | **Apache Commons Lang3** | **建议补充**。用于字符串判空、随机数生成（生成预约号/ID）等，避免重复造轮子。 |

## 三、 前端技术

| 工具/库 | 名称/版本 | 说明 |
| :--- | :--- | :--- |
| **核心技术** | **JSP + JSTL** | **补充**。JSP 负责写页面，JSTL（Jakarta Standard Tag Library）负责在页面上循环显示数据（如科室列表）。多用标签库。 |
| **UI 框架** | **Bootstrap 5** | **补充**。**不用写 CSS！** 直接引用 Bootstrap 的 CDN 链接。它提供了现成的“导航栏”、“表格”、“表单”、“按钮”样式。 |
| **图标库** | **Bootstrap Icons** | **补充**。为你的按钮增加“删除”、“修改”等小图标。 |
| **交互脚本** | **原生 JavaScript** | 既然是简单交互，不需要 React/Vue，简单的表单验证用原生 JS 即可。 |

## 四、 特定功能组件

| 功能 | 工具/库 | 说明 |
| :--- | :--- | :--- |
| **Excel 处理** | **EasyExcel** | 阿里的库，一行代码搞定读写，比 Apache POI 简单。 |
| **PDF 生成** | **OpenPDF** (LibrePDF) | **补充推荐**。它是 iText 的开源分支，简单易用。用于生成月度报表。**Maven 依赖搜：`com.github.librepdf:openpdf`**。 |
| **文件上传** | **Servlet 6.0 自带** | Tomcat 11 支持 Servlet 6.0，自带 `Part` 接口处理文件上传，**不需要**额外的 `commons-fileupload` 包了。 |

## 五、 部署与运维

| 工具 | 说明 |
| :--- | :--- |
| **服务器 OS** | **Ubuntu 24.04 Server** (你已选定)。记得配置防火墙 (UFW) 开放 8080 端口。 |
| **终端工具** | **Termius** (Mac/Win)。用于 SSH 连接服务器。 |
